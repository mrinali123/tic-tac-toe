# Tic-Tac-Toe

A clean, browser-based 2-player Tic-Tac-Toe game built with HTML, CSS and JavaScript.

## Features
- Enter custom player names before starting
- Displays whose turn it is live
- Detects winner and draw automatically
- Reset / New Game support

## How to Play
1. Enter names for Player X and Player O
2. Click START and play!

## Tech Stack
- HTML5
- CSS3
- JavaScript
